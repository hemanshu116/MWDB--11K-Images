## Installation

System Requirements: python version of 3.3 or more should be installed

The following packages must be installed before the prior execution of the python file
• pip install numpy
• pip install skimage
• pip install opencv-python
• pip install scikit-learn
• pip install matplotlib
• pip install pandas

## Execution
Run the python file on commandline using the following command
> python gui.py
> select any of the tasks to run bases on choice
> Select inputs based on tasks selected

## Output
The outputs are extracted in the DatabaseFolder in the current working directory.
